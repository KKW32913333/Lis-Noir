# Lis Noir - CARD BATTLE

ドラゴンと聖女のゴシックファンタジー世界観を舞台にした、ブラウザで動くカードバトルゲーム（PWA対応）。

> 現在の状態：**プロトタイプ（コア機能が一通り遊べる段階）**

---

## 1. 技術スタック

- HTML / CSS / JavaScript のみ（ビルドツール・フレームワーク不使用）
- ブラウザでそのまま動作し、GitHub Pagesで無料公開可能
- 保存データは `localStorage`（ブラウザ内保存）
- フォント：Google Fonts（Cinzel / Noto Serif JP）をCDN読み込み


## 4. カードの追加・差し替え方法

`cg-game.js` の `CARD_DEFS` にカード情報が1箇所にまとまっています。

```js
fire_dragon: {
  name: 'ファイアドラゴン', element: 'fire', rarity: 'legend',
  cost: 5, atk: 6, hp: 10, skill: '攻撃時、敵全体に2ダメージ',
  image: null,        // ← ここにファイル名を入れると画像が反映される（例: 'card-fire-dragon.png'）
  emoji: '🐉'          // image が null の間だけ使われるプレースホルダー
}
```

画像素材を追加する場合の流れ：
1. 白背景・被写体中心のプロンプトでAI画像生成
2. 生成画像をアップロード → 背景除去・トリミング・サイズ統一して加工
3. `image` フィールドにファイル名を登録

## 4.5 ショップの仕組み

`cg-game.js` の `SHOP_PACKS` にパック定義がまとまっています。

```js
{ id: 'rare', name: 'レアパック', icon: '🎁', currency: 'gems', cost: 10,
  desc: 'レア以上が確定で出るパック', weights: { normal: 0, rare: 65, epic: 28, legend: 7 } }
```

- `currency`: 消費する通貨（`gold` または `gems`）
- `weights`: レアリティごとの抽選重み（0にするとそのレアリティは出ない）
- 購入すると、重みに従って対象レアリティからカードをランダム抽選 → そのカードの経験値+20（100で1レベルアップ）
- 全カードは最初から所持済みという設計のため、パックは「新規解放」ではなく「強化素材（経験値）を引く」仕組みにしています

## 4.6 ミッションの仕組み

`cg-game.js` の `MISSIONS` に定義がまとまっています。

```js
{ id: 'win3', title: '勝利を重ねる', desc: 'バトルに3回勝利する', target: 3,
  check: s => s.totalWins || 0, reward: { gold: 500 } }
```

- `check`: 現在の進捗を state から算出する関数（勝利数・パック開封数・強化回数・デッキ枚数など）
- `target` に到達すると「受け取る」ボタンが有効化され、タップで `reward` の分だけゴールド/ジェムを付与
- 受け取り状況は `state.missionsClaimed` に保存され、再度は受け取れない
- 現在は全て「累計〇〇回」の実績型ミッション。日次リセット型のデイリーミッションは未実装（ホーム画面のデイリー報酬バーは現状まだ装飾のみ）

## 4.7 スペル/装備カードの仕組み

`CARD_DEFS` の各カードに `type`（`monster`/`spell`/`equipment`、省略時は`monster`）と `target`、`effect` を持たせています。

```js
spell_fireball:  { name: 'ファイアボール', type: 'spell', target: 'enemy',
  effect: { kind: 'damage', value: 4 }, cost: 2, ... }
equip_shield:    { name: 'ガーディアンシールド', type: 'equipment', target: 'friendly',
  effect: { atk: 0, hp: 4 }, cost: 2, ... }
```

**操作方法**
- スペル（`target: 'enemy'`）: 手札で選択 → 敵モンスターまたは敵本体をタップして発動
- スペル（`target: 'none'`）: 手札でタップした瞬間に即発動（回復・ドロー等、対象選択不要）
- 装備（`target: 'friendly'`）: 手札で選択 → 自分の場のモンスターをタップして装備。攻撃力/HPが戦闘計算に加算される

**現在の制限（既知の制約）**
- 敵AIはスペル/装備カードを使用しません（モンスターの配置・攻撃のみ）
- 装備の攻撃力ボーナスは戦闘計算には正しく反映されますが、フィールド上のミニカード表示には現状バッジ表示されません（HP増加分はHPバッジに反映されます）
- フィールドカード（docxで言及されている常時効果カード）は未実装です

## 5. 画面配色・デザイン方針

添付いただいた「Lis Noir」アプリアイコンから着想した、中間の明るさのテーマ「**ドーンゴールド**」を採用。生成り寄りのタウプ背景(#DDD3C0)に、ゴシックな重厚感を残すダークパネル(#2A2015)を組み合わせ、ゴールド(#A97A28)・ワインレッド(#7A2536)・紫(#6B4FA0)をアクセントに使用。フォントもNoto Serif JP中心のserif書体で世界観を統一。
